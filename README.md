# ERP Student Course System Project  

# Author  
Adabala.Sai Prasad Mavullamma  

---

# Description  
A menu-driven ERP system for managing students, courses, and course registrations using Python.  
The project follows modular programming using functions and in-memory data structures (dictionaries).

---

# Features  
- Student Management  
- Course Management  
- Course Registration  
- View Students List  
- View Courses List  
- View Registration Details  

---

# Technologies Used  
- Python  
- Dictionaries (In-memory Data Storage)  
- Functional Programming
